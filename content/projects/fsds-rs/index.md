## FSDS Rust API

The ['fsds-rs' crate](https://crates.io/crates/fsds-rs) provides users a Rust API to create clients that can connect with the [Formula Student Driverless Simulator](https://fs-driverless.github.io/Formula-Student-Driverless-Simulator/v2.2.0/).

It is a copy of the official [Python API](https://fs-driverless.github.io/Formula-Student-Driverless-Simulator/v2.2.0/getting-started-with-python/).

It was developed to test the first version of the UniUD E-Racing team first implementation of the driverless pipeline.
